## Variables que debes reemplazar antes del envío

- `[NOMBRE]`
- `[NOMBRE PET SHOP]`
- `URL_LINK_AGENDAR`, si usarás un link de reunión diferente al WhatsApp de prueba.

## Recomendación

Para Gmail personal, lo más seguro es:
1. Subir assets a GitHub Pages.
2. Reemplazar las rutas locales por URLs públicas.
3. Abrir el HTML en navegador.
4. Copiar visualmente el render del navegador.
5. Pegar en Gmail.
6. Enviar pruebas antes del cliente.
